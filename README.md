# Setinela - POC Redis Replication

Just a example Rails App to test Redis with failover and [replication](http://redis.io/topics/replication), using [redis-sentinel](http://redis.io/topics/sentinel)

## System dependencies
* [Redis](http://redis.io)
* [Vagrant](http://vagrantup.com)
* [vagrant-dns plugin](https://github.com/BerlinVagrant/vagrant-dns)
* [VirtuaBox](https://www.virtualbox.org/)
